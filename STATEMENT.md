PROBLEM STATEMENT

Buying a house is a big decision, and getting the price right is tough . Factors like location, amenities, and population density all play a role, but it's hard to weigh them accurately. 
Current estimates often fall short, leading to poor investment choices. This project tackles this issue by creating a straightforward machine learning model that predicts house prices using key factors,
helping buyers and investors make informed decision



OBJECTIVE
The main objective of this project is to:
- Develop a predictive model for house prices by applying machine learning concept
- Provide a simple tool for estimating property values
